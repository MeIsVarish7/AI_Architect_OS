# Projects

Purpose:

Convert knowledge into proof.

This folder tracks every software, AI, research, and engineering project built while following AI Architect OS.

Projects are NOT organized into Active, Completed, or Showcase folders.

Instead, a single dashboard tracks the status of every project.

This allows any AI manager to instantly understand:

- What is being built.
- What is planned.
- What is paused.
- What is completed.
- What should be worked on next.

The dashboard is the single source of truth.

The goal is not to build many projects.

The goal is to build projects that demonstrate increasing engineering ability.
