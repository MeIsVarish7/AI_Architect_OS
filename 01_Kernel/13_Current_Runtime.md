# Current Runtime

## Purpose

The Current Runtime stores temporary information required only for the current learning session.

Unlike the rest of AI Architect OS, this file is expected to change before every session.

It represents the runtime memory of the Operating System.

Permanent knowledge belongs elsewhere.

Temporary session information belongs here.

---

## Primary Mission

Provide today's runtime context so the Operating System can generate the most effective learning session.

## AI Instructions

Before generating a Dynamic Session Prompt:

- Read this file.
- Combine it with the permanent Operating System.
- Adapt today's session accordingly.
- Do not overwrite permanent learning progress using temporary runtime information.

---

## Constraints

This file should never contain:

- Curriculum Progress
- Current Level
- Current Topic
- Completed Topics
- Projects History
- Research History

Those belong to the permanent Operating System.

---

## Success Criteria

A successful Current Runtime should provide enough temporary context to personalize today's learning session without duplicating permanent information.
