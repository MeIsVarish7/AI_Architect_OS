# AI Architect OS Implementation

## Purpose

This folder contains the reference implementation of AI Architect OS.

The Kernel defines the architecture.

The Curriculum defines the knowledge.

The Implementation brings the Operating System to life.

The goal is to transform the AI Architect OS specification into a working application.

---

## Development Philosophy

Implement only what has already been specified.

Do not invent new architecture while coding.

If implementation reveals architectural problems:

1. Stop.
2. Audit.
3. Update the specification.
4. Continue implementation.
